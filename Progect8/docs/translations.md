# translations.py

## Призначення
Локалізація текстів додатка (англійська і українська). Надає механізм `t('key')` для шаблонів.

## Очікуваний вміст
- змінна `TRANSLATIONS`: словник видами `{ 'en': {...}, 'ua': {...} }`
- функція `get_text(key, lang='en')`: повертає рядок перекладу або ключ, якщо переклад відсутній

## Використання в app.py
- Контекстний процесор `translation()` додає `t` у шаблони:
  - `t('hello')` → локалізована фраза
  - `lang` → поточна мова

## Модифікації
- Додати нові фрази для нових UI-елементів (status, admin flow) в `TRANSLATIONS`.
