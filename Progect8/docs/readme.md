# Проєкт Adaptive Difficulty Tournament System

Це технічна документація, яка описує кожен основний файлу.

- `app.md` - детальний опис маршрутової логіки і контролерів.
- `models.md` - опис моделей SQLAlchemy.
- `utils.md` - адаптивні функції для розрахунку складності.
- `translations.md` - локаль, фрази і i18n-механіка.
- `demo_setup.md` - як підняти демо-дані.

## Як користуватися
1. Перейдіть у `docs/`.
2. Відкрийте потрібний `*.md`.
3. За потреби доповніть власний `API_ukr.md` для endpoints/params.
