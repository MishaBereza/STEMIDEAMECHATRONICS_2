Ось повний опис кожного основного файлу/папки в цьому проекті (українською):

- `app.py` - головний файл Flask-додатку. Тут створюється екземпляр додатку, ініціалізується база даних, визначаються всі маршрути (роути) для користувачів, команд, турнірів і адміністратора. Також тут оброблюється логін адміністратора, панель адміна та функції для реєстрації, подачі заявок і роботи з даними.

- `models.py` - моделі SQLAlchemy, що описують структуру бази даних. Містить класи: User, Tournament, Team, Round, Submission, Evaluation з їхніми полями і зв'язками.

- `utils.py` - допоміжні функції для адаптивної складності раундів. Обчислюється середня оцінка, коригується рівень складності та генерується об'єкт нового раунду.

- `demo_setup.py` - скрипт для створення демонстраційних даних: адміністратор, суддя та простий турнір з одним раундом. Запускається вручну або через кнопку в інтерфейсі.

- `requirements.txt` - перелік Python-залежностей, які необхідно встановити через pip.

- `admin_password.txt` - файл з автоматично згенерованим паролем для входу в адмін-панель.

- `start.bat` і `start.ps1` - допоміжні батники/PowerShell скрипти для швидкого запуску додатку на Windows.

- Папка `templates/` - всі HTML-шаблони проекту. Вони використовують Jinja2 для рендерингу. Основні шаблони:
    * `base.html` - базовий шаблон з навігацією, переключенням мови і обробкою флеш-повідомлень.
    * `index.html` - головна сторінка з кнопками реєстрації, створення демо-даних та списком турнірів.
    * `register.html` - форма реєстрації нового користувача.
    * `tournament.html` - сторінка деталей турніру і раундів.
    * `leaderboard.html` - таблиця результатів для турніру.
    * `profile.html` - особистий профіль користувача.
    * `admin_login.html` - форма входу для адміністратора.
    * `admin_dashboard.html` - панель адміністратора з показниками (користувачі, турніри тощо).
    * `admin_users.html` - сторінка керування користувачами (зміна ролей).

- Папка `templates/scripts/` або `templates` додаткові шаблони, які відповідають за інші частини інтерфейсу (залежно від структури, згадані раніше).

- Папка `tests/` - тести для проекту. Зараз містить `test_utils.py` з юніт-тестами функцій у `utils.py`.

- `translations.py` - словники для багатомовності (англійська/українська) для інтерфейсу.

- Інші файли/папки (`__pycache__/`, `instance/`) є стандартними для Python/Flask і використовуються для скомпільованих файлів та конфігурації/бази даних.

Цей опис можна використовувати як довідку по структурі проекту.
